Coding Summer School

During this event, participants are learning to build a website, a game and to deploy their work. Various presentations from professionnals are also explaining the goal of learning code and how it is important in their work.

Your weapons to build an awesome website

In order to write some code, we use programming languages.
For your website, you will need to learn 3 programming languages.
You will learn them throughout some videos that will show you different possibilities and will help you to do so step-by-step!

  • HTML - used for defining the content of your website

  • CSS - used for defining the style of the elements of your website

  • JS - used for interactivity / making games / ... on your website
